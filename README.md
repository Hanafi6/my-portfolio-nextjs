# 💼 My Portfolio – Next.js

مرحبًا 👋  
ده البورتفوليو الشخصي بتاعي، معمول باستخدام **Next.js 14**، وبيستخدم **Tailwind CSS** و**Framer Motion** لواجهة أنيقة وسلسة.

---

## ✨ المميزات

- ⚡️ تصميم متجاوب تمامًا (Responsive Design)
- 🎯 تنقل سلس باستخدام `framer-motion`
- 🧠 إدارة الحالة بـ Zustand
- 🧩 مكونات قابلة لإعادة الاستخدام
- ⚙️ بنية منظمة واحترافية للمشروع

---

## 🛠️ التقنيات المستخدمة

- [Next.js 14](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
- [Zustand](https://github.com/pmndrs/zustand)
- [Lucide Icons](https://lucide.dev/)
